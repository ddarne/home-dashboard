# Project Name

Home Dashboard:
- Weather App
- MTA Trains

## Installation

- Install JDK 1.8 or later
- Install Gradle 2.3+ or Maven 3.0+

## Usage

- Clone repository
- ./gradlew build
- java -jar build/libs/dashboard-web-content-0.1.0.jar
- Go to http://localhost:8080
